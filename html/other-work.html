Artivox (2024)

Software Developer (2020-Present)

(C++, JUCE, AWS: EC2/RDS/VPC/Lambda, Stripe API)

About

Artivox is an application that synthesizes a human voice for singing or speaking, allowing the user to create their own voices without needing to sample or record a human voice.

It was originally created as an alternative to the popular Vocaloid products, where the designer could make whatever voice they wanted, instead of buying voice packs.

Eventually, though, Artivox took on an identity of its own, with the intention of replicating any known (or unknown) instrument.

It is still in an elementary phase, that I continue to work on in the background with more engineering experience, but I do use for a lot of synth work.

Usage

While intended for music purposes, it can be used for a variety of sound making purposes.

Worked in conjunction with another tool, Artitalk, a conversational chatbot manager that used NLP through Python scripts to send text for Artivox to speak.

It also is intended to work with my lofi music generator, LVCK, which gives Artivox the compositional logic to create music.

The Little Piano (2021)

Designer (2021)

(PureData, Python, Bash, EE)

About

The Little Piano is a minirature piano that generates it's own music, using PureData on a Raspberry Pi 3.

It was a predecessor for my lofi music generator, LVCK, but given a physical model to emulate actual acoustics (like a lid to dampen the sound).

The Little Piano was a small gift for the Assistant Dean of Bienen School of Music, who helped greatly with my academic scheduling.

I have a demo video explaining the process here (youtube.com).

Gitra (2024)

Software Developer (2024)

(C++, Git, Git LFS)

About

Gitra is a small program that combines the 3DS emulation of Citra with the source control of Git, hence the name "Gitra".

Arose from a need to share save files with Citra for Animal Crossing: New Leaf, Gitra will make sure all players are up to date.

Acts as one-click to play, where it executes exclusive checkouts, manages the application, and keeps all players up to date with the latest save files.

This is just a fun project I thought I'd include, instead of all the super technical work.

While a lot of personal work is trying to artificially replicate something, sometimes I use my tools for dumb little projects to share with friends.

You can find the Github repository and builds here (github.com).
